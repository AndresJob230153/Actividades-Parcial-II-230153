# Actividades Parcial II - Automatización de Infraestructura Digital I

Repositorio que reúne las actividades desarrolladas durante el **Segundo Parcial** de la asignatura **Automatización de Infraestructura Digital I** de la carrera **Ingeniería en Redes Inteligentes y Ciberseguridad** de la **Universidad Tecnológica de Aguascalientes**.

El objetivo de este repositorio es mantener organizadas las prácticas realizadas durante el curso, aplicando herramientas de control de versiones, documentación técnica y automatización de tareas mediante diferentes tecnologías empleadas en la administración de infraestructura digital.

# Descripción

Este repositorio contiene las actividades desarrolladas durante el segundo parcial de la materia **Automatización de Infraestructura Digital I**.

Las prácticas tienen como finalidad fortalecer las competencias relacionadas con la automatización de procesos, documentación técnica, control de versiones y desarrollo de scripts utilizando Python y otras herramientas ampliamente utilizadas en el ámbito de la infraestructura digital.

Entre los temas abordados se encuentran:

- Elaboración de documentación profesional mediante Markdown.
- Uso de Git y GitHub para control de versiones.
- Desarrollo de pruebas unitarias utilizando `unittest`.
- Manipulación de archivos en formato JSON.
- Manipulación de archivos YAML.
- Organización y documentación de proyectos.
